
# Smarter Blinkit

AI-powered grocery marketplace connecting buyers with nearby sellers.

## Features

- Dual login system (Buyer / Seller)
- Intent-based smart search
- Barcode based inventory system
- Recipe based shopping assistant
- Smart order splitting
- Product recommendation system

## Tech Stack

Frontend: React
Backend: Node.js
Database: PostgreSQL
Graph Database: Neo4j
Payments: Razorpay Test API

## Project Structure

frontend -> user interface
backend -> server and APIs
database -> database schema
docs -> documentation
